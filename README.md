# Qwixx

A compact installable scorekeeper for Qwixx. It is a static React app designed for GitHub Pages and local device persistence.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## GitHub Pages

The included workflow deploys `dist` to GitHub Pages on every push to `main`. In GitHub repository settings, set Pages to use GitHub Actions.
